Bash Basics Review


Introduction

This lesson will offer a brief overview of bash and review from the Bash Basics course along with some additions. Bash has a man page, which can be viewed with man bash. Bash has many default behaviors, which determine how bash interprets the command line sent to it. This man page should be periodically reviewed wherever possible and is invaluable when you're not quite sure about the unexpected token error that just popped up. Remember: knowledge of how bash operates and interprets saves lives.


Bash Review, Documentation, and Invocation

Bash is an sh-compatible command language interpreter whose functionality hinges on the use of I/O streams. Its function is to interpret strings passed to its input stream, execute the appropriate programs, and then direct the result as its output stream.
